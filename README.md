# Изучение Pandas и NumPy

## Файлы:

1. `pandas/guide/` - директория с туториалами **Pandas** по видеоурокам
2. `pandas/practice/urfu_SaVD` - директория с решением заданий на **Pandas** по курсу сбора и верификации данных
3. `pandas/practice/UrFU_ADaAI_contest` - директория с решением заданий на **Pandas** с контеста по курсу анализа данных и искуственного интеллекта
4. `numpy/guide/` - директория с туториалом **NumPy** по видеоуроку
5. `numpy/practice/` - директория с решением заданий с контекста по **NumPy** по курсу анализа данных и искуственного интеллекта
